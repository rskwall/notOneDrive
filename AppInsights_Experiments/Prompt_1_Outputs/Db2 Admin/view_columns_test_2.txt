## Steps

1. At the **Option ===>** prompt, type `1` and press **Enter** to open the **System Catalog - Objects** menu.

2. In the **Name** field, type `[Database Name Pattern]` (e.g., a partial name using a wildcard such as `ABC%`).

3. At the **Option ===>** prompt, type `D` and press **Enter** to display the **Databases** list.

4. In the **Select** column next to the target database entry, type `T` and press **Enter** to open the **Tables, Views, and Aliases** list for that database.

5. In the **Sel** column next to the target table, type `C` and press **Enter** to view the **Columns** for that table.

6. Review the column list, which displays:
   - Column Name
   - Column Number
   - Column Type
   - Length
   - Scale
   - Nullable indicator
   - Default indicator
   - Column Cardinality

7. When finished reviewing, press **PF3** to return to the **Tables, Views, and Aliases** list.

8. Press **PF3** again to return to the **Databases** list.

9. Press **PF3** again to return to the **System Catalog - Objects** menu.

10. Press **PF3** again to return to the **DB2 Admin Main Menu**.
